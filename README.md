**自主HTTP服务器**

- 项目描述：开发HTTP服务器，完成请求响应模型的交互；

- 项目核心功能：实现静态网页资源返回，动态网页交互过程；

- 使用技术：HTTP协议、Socket编程、文件操作、线程池、Fiddler使用；

- 项目特点：采用线程池搭建HTTP服务器，提高并发效率；

　　　　　       客户端能够使用GET，POST方法请求资源，支持多种格式如HTML、JPG、PNG等；

　　　　　       支持差错处理，客户端请求的资源不存在时，服务器能够返回一个404页面；
